# CloudExctrators

A website that let's you scrape information from various sites for leads. 

# Programming

The projects is coded fully in PHP using various symphony modules.
Some scraping sources required simple http requests while some required headless browsing.

# What I learned

- A more in-depth understanding of REST APIs. 
- Web crawling optimisation by loading only parts we need.
- I improved my skills with a couple of jquery plugins like dataTables and graphs.
